# Beauty за минути — EUR↔BGN Ресто Калкулатор

Този малък статичен сайт предоставя лесен калкулатор за пресмятане на ресто при преход от BGN към EUR.

Характеристики:
- Двупосочно синхронизиране: въвеждане на стойност в EUR автоматично попълва стойността в BGN и обратно (за цена и за внесена сума).
- Ресто (change) се изчислява автоматично, като се показва и в EUR и в BGN.
- Закотвен курс: 1 EUR = 1.95583 BGN.

Файлове:
- `index.html` — основна страница
- `css/style.css` — стилове
- `js/script.js` — логика за пресмятане и синхронизация
- `netlify.toml` — минимална Netlify конфигурация

Локален преглед:
Отвори `index.html` в браузър (двоен клик) или стартирай прост HTTP сървър от папката проекта, например с Python3:

```bash
python3 -m http.server 8000
# и след това отвори http://localhost:8000
```

Публикуване в Netlify:
1. Създай нов сайт в Netlify чрез GitHub/Bitbucket/GitLab или drag-and-drop на тази папка (build command: празно, publish directory: ".").
2. Netlify ще служи `index.html` автоматично.
